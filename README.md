# Minecraft Chinese Localization Tool

Windows-first desktop workbench for creating Chinese localization patches for Minecraft modpacks.

This app helps scan a modpack, extract translatable strings, pre-translate them with a pluggable backend, review them manually, and export a patch zip without modifying the original modpack.

## Features

- Visual Electron + React desktop UI.
- Supports `zh_cn` and `zh_tw` projects.
- Imports local instance folders and `.zip` / `.mrpack` archives.
- Detects common launcher instances: PCL2-style `.minecraft`, Prism/PolyMC, CurseForge, Modrinth App.
- Scans:
  - `mods/*.jar` Minecraft lang files.
  - `resourcepacks` lang files.
  - `config/ftbquests`, `config/betterquesting`, `config/hqm`.
  - `kubejs` scripts and JSON files.
  - Patchouli guidebook JSON inside mod jars.
  - advancement JSON and shader text/lang files.
- Manual translation workbench with search, source filter, status filter, notes, AI translation, and human-reviewed translation columns.
- CSV/XLSX export and import for external translators.
- Project glossary / translation memory used by AI prompts.
- Translation backend adapters:
  - Manual mode.
  - Ollama/local model API.
  - OpenAI-compatible `/chat/completions` API.
- Exports a patch zip containing `resourcepacks/`, `config/`, `kubejs/`, `datapacks/`, and other transformed files where applicable.

## Project Files

The app stores project state under `.mclocalizer/`:

- `project.json`: source path, workspace path, target locales, Minecraft version, loader, scan summary.
- `strings.jsonl`: extracted translation table, one string per line.
- `glossary.json`: project glossary and translation memory.
- `output/`: generated patch folders and zip files.

Archive imports are extracted into the user's Documents folder under:

```text
Documents/Minecraft Chinese Localization Tool/Projects/
```

## Patch Output

The app does not edit the original modpack. It produces a patch zip. Copy the patch contents into the target instance.

Resource packs generated by this app should usually be enabled below broad automatic localization packs, so reviewed patch strings override missing or corrected translations.

## Notes

- AI translation is only a draft. The intended workflow is extract -> pre-translate -> human review -> export patch.
- Keep placeholders, formatting codes, item IDs, and Minecraft color codes unchanged during manual review.
- Older Minecraft versions may prefer `.lang` files; modern versions use JSON lang files. The exporter preserves the scanned language file format where possible.
